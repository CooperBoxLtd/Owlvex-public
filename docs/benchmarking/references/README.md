# Benchmark References

This directory is the reference-data layer of the Benchmarking Department.

Purpose:

- store mappings between Owlvex and external benchmarks
- document what external benchmark slices are in scope
- avoid fuzzy references like "we use OWASP Benchmark" without saying how

Reference documents in this directory should answer:

- which external benchmark is being referenced
- what exact slice or subset Owlvex cares about
- which Owlvex issue families map to it
- what is intentionally excluded
