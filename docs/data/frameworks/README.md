# Curated Framework Blobs

This directory stores Owlvex-authored framework blobs derived from raw upstream framework material under [framework-sources](D:/Dev/repos/CodeScanner/docs/data/framework-sources).

Why blobs:

- the current product does not need a database for framework guidance
- versioned JSON blobs are easier to diff, review, ship, cache, and sign as rule-pack artifacts
- the AI lane benefits from compact curated summaries rather than raw standards dumps

Current blob:

- [owlvex.framework-pack.2026.1.json](D:/Dev/repos/CodeScanner/docs/data/frameworks/owlvex.framework-pack.2026.1.json)

Design intent:

- keep upstream raw sources separate from Owlvex-curated prompt guidance
- preserve provenance so each framework profile can be traced back to official or reference sources
- feed future prompt-pack and framework-pack delivery without requiring a database dependency
