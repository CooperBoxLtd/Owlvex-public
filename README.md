# Owlvex

Public distribution repository for the Owlvex production VS Code extension.

## Current release

- Version: 0.8.0
- Package: `downloads/owlvex-prod-0.8.0.vsix`
- Marketplace: https://marketplace.visualstudio.com/items?itemName=owlvex.owlvex

## Install from VSIX

```powershell
code --install-extension .\downloads\owlvex-prod-0.8.0.vsix
```

Owlvex scans source code locally, supports AI-assisted review when configured by the user, creates evidence reports, and opens reviewable fix previews before changing files.
