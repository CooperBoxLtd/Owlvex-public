# Support

For Owlvex support, open a GitHub issue:

https://github.com/CooperBoxLtd/Owlvex-public/issues/new

You can also contact support by email:

info@cooperbox.co.uk

Use issues for:

- installation problems
- extension activation errors
- scan failures
- incorrect or missed findings
- confusing reports
- fix preview problems
- AI provider setup problems

Please include:

- operating system
- VS Code version
- Owlvex version
- command you ran
- visible error message
- whether the issue happens with deterministic scanning, AI scanning, or both
- AI provider, if AI scanning is involved

Do not include API keys, secrets, proprietary source code, or customer data in public issues.

For private or sensitive support requests, email info@cooperbox.co.uk instead.
